Commands to Add All Website Files to Git
==========================================

Run these commands in your terminal:

# HTML Pages (9 files)
git add index.html team.html robot.html competitions.html outreach.html gallery.html sponsors.html join.html contact.html

# Stylesheet
git add styles.css

# JavaScript files
git add script.js js/nav-footer.js

# Logo files
git add RoboPhantomLogo.png FtcFirstLogo.jpeg

# Image directories
git add TeamPictures/ Coaches/ Robot/ Sponsors/

# .gitignore (optional but recommended)
git add .gitignore

# Verify what was added
git status

# Commit the files
git commit -m "Add all website files"

---

OR use the simple command (will respect .gitignore):

git add .

This will add all files except those in .gitignore:
- All .md files are excluded
- Development scripts are excluded
- Backup files are excluded

Then check what will be committed:
git status
